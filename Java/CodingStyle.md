## 编程风格

### Allmans风格

“独行”风格，左右大括号各占一行。

如：

```
public class Allmans
{
	public static void main(String args[]) 
	{
		System.out.println("Hello Allmans!");
	}
}
```

### Kernighan风格

“行尾”风格，左大括号在上一行的行尾，而右大括号独占一行。

如：

```
public class Kernighan {
	public static void main(String args[]) {
		System.out.println("Hello Kernighan!");
	}
}
```
