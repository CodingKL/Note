## 简介

- 是一种在 Java 虚拟机上运行的静态类型编程语言。
- 由 [JetBrains](https://baike.so.com/doc/5381594-5617929.html)设计开发并开源。 
- 可以编译成Java字节码，也可以编译成 JavaScript，方便在没有 JVM 的设备上运行。 
- Android官方开发语言 。
- 程序文件以 .kt 结尾。

## 第一个 Kotlin 程序

```
fun main(args:Array<String>){
	println("hello kotlin")
}
```
