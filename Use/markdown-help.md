# 帮助

**快捷键**：

- 粗体：Ctrl+B
- 斜体：Ctrl+I
- 删除线：Ctrl-S
- 标题：Ctrl-H
- 代码块：Ctrl-Alt-C
- 引用：Ctrl-Q
- 横线：Ctrl--
- 无序列表：Ctrl-L
- 有序列表：Ctrl-Alt-L
- 链接：Ctrl-K
- 图片：Ctrl-Alt-I
- 表格：Ctrl-Alt-T
- 预览模式：Ctrl-P
- 全屏模式：F11
- 边写边看：F9
- 清除格式：Ctrl-E



注：

- 支持缓存到使用的浏览器中（在不同浏览器是不同步）。


- 该Markdown在线编辑器使用了开源项目[simplemde-markdown-editor](https://github.com/sparksuite/simplemde-markdown-editor)进行自定义改造的。
- 此编辑器只为更好的写作。
